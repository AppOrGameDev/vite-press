      Send {Ctrl down}{LWin down}{Left}{LWin up}{Ctrl up}

      ^+Right::

      return

    - 前置准备`ctrl`+`win`创建两个虚拟桌面,左(1)放日常办公本地桌面;右(1)放远程桌面连接

    - 上面的脚本+`AutoHotKey`达到的效果是

      - `ctrl`+`shift`+`方向右键`=向右切换到远程桌面连接
